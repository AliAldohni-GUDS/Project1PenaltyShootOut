# Penalty Shootout – Processing Game

This project was created for Coding Camp Project 1 using Processing (Java mode).

## How to run
1. Open Processing (Java mode)
2. Open `PenaltyShootout.pde`
3. Click Run

## Controls
### Menu
- 1 / 2 / 3 → Select difficulty (Easy / Normal / Hard)
- ENTER → Start game

### In Game
- LEFT / RIGHT → Aim
- UP / DOWN → Power
- SPACE → Shoot
- R → Next shot
- M → Back to menu
- X → Full reset

## Features
- Probability-based goal vs save
- Difficulty levels
- Goalkeeper dive animation
- Object-oriented design with multiple classes
